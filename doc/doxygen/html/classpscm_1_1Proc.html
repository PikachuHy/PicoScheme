Detailed Description

Procedure type to represent a scheme closure. 

* (lambda args body)  => closure: [symenv, args, body]
*
*  with:
*  args        := [symbol | nil | list | dotted-list]
*  list        := nil | (expr . list)
*  dotted-list := (list . expr)
*  body        := (expr . list)
*
