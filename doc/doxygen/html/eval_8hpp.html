Scheme begin syntax to evaluate a sequence of scheme expressions in consecutive order.

Evaluate each expression in argument list up the last, which is returned unevaluated.

Returns
Unevaluated last expression or special symbol none for an empty argument list.

This last expression is evaluated at the call site to support unbound tail-recursion.
