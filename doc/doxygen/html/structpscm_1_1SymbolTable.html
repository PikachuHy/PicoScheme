Detailed Description

template<typename T, typename Hash = std::hash<T>, typename Equal = std::equal_to<T>>
struct pscm::SymbolTable< T, Hash, Equal >

Symbol table of to provide unique symbols.

A symbol table is bijective mapping between values of type T to unique symbols of type Symtab<T>::Symbol.

Template Parameters
TValue type of symbol, like std::string, char, int,...
HashHash function object that implements a has function for values of type T.
EqualFunction object for performing comparison on values of type T.
