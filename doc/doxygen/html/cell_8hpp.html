T &&  t,
Args &&...  args 
)

Build a cons list from all arguments directly in in argument cons cell array.

This array embedded cons-list is used for short temporary argument lists to circumvent to unecessarly fill the global cell store.

The cons array size must be equal or greater the number of remaining arguments. An insufficient array size is an compile time error.
